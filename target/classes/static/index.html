使用说明

工作流程：

  1. 添加订单 - 将抖音平台的客户订单录入系统，包括收件人信息
  2. 淘宝采购 - 在淘宝向供应商下单，获取快递单号
  3. 面单转换 - 将淘宝的物流信息转换为抖音平台格式，完成发货回传
  4. 轨迹同步 - 将淘宝的物流轨迹实时同步到抖音平台

提示：物流映射功能用于配置淘宝物流公司与抖音平台物流公司的对应关系，系统已预置常见快递公司映射。
